# cub3d
Mini game-engine on raycast graphic

**Bonuses completed**
- Wall collisions
- An HUD
- Ability to look up and down
- Rotate the point of view with the mouse

**OS**
- Linux Ubuntu

**Compilation**
- make re

**Run**
- ./cub3D map.cub

**Take a screanshot**
- ./cub3D --save map.cub

**Control keys**
- Esc - exit
- WASD - moving
- H - include mouse
- M - turn on minimap
- Arrows or mouse - Rotate the point of view
